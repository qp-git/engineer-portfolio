# 公開ポートフォリオ概要

このリポジトリは、クラウド・インフラ領域の学習成果や個人開発プロジェクトを整理するための公開ポートフォリオです。

## 目的

転職活動や技術説明の場で、自分が学習・検証した内容を外部から確認できる形に整理することを目的としています。

## 掲載方針

- 社内研修や業務に関する機密情報は掲載しない
- 公開可能な個人開発・検証内容を中心に整理する
- 構成、使用技術、工夫した点、学びを簡潔にまとめる
- 面接で説明しやすいよう、技術選定や設計意図も記録する

- OdaiBox
- STT + ECS
- Alexa Skill
- Interview Knowledge Bridge
