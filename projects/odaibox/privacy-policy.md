# OdaiBox Privacy Policy

OdaiBox は、Discord サーバーでゲーム用のお題をランダムに出す Bot です。

このページでは、OdaiBox が保存する情報、保存しない情報、情報の利用目的を整理します。

## 基本方針

OdaiBox は、Bot の機能提供に必要な範囲に限って情報を保存します。

保存する情報は、サーバーごとのお題管理、同じお題の連続回避、管理画面へのログイン制御などに利用します。

不要な情報は取得・保存しません。

## 保存する情報

OdaiBox では、主に以下の情報を保存します。

| 情報 | 利用目的 |
|---|---|
| Discord サーバー ID | サーバーごとにお題設定を分けるため |
| Discord ユーザー ID | お題履歴や管理操作の記録に利用するため |
| お題データ | サーバーごとのカスタムお題を管理するため |
| お題の有効 / 無効状態 | 出題対象を制御するため |
| 一時的なお題履歴 | 同じユーザーに同じお題が続きにくくするため |
| `/admin-login` の発行情報 | 管理画面ログイン用の一時パスワードを制御するため |

ユーザー ID やサーバー ID は、Discord 上で人やサーバーを区別するための識別子です。

パスワードやクレジットカード番号のような秘密情報ではありませんが、利用者やサーバーの行動をたどる手がかりになり得るため、必要な範囲に絞って扱います。

## 保存しない情報

OdaiBox は、以下の情報を保存しません。

- Discord の音声通話
- チャット本文
- DM の内容
- Discord のパスワード
- メールアドレス
- 支払い情報
- 位置情報
- 端末内のファイル

OdaiBox は、スラッシュコマンドや管理画面の操作に必要な範囲で動作します。

音声通話の内容や、通常のチャット本文を記録する目的の Bot ではありません。

## 管理画面ログインについて

OdaiBox では、管理画面にログインするために `/admin-login` コマンドを利用します。

権限のあるユーザーが `/admin-login` を実行すると、Bot が一時パスワードを発行します。

一時パスワードは、対象の Discord サーバーと紐づけて保存されます。

この仕組みにより、固定の共通パスワードを配布せず、Discord 側の権限確認を利用して管理画面へのログインを制御します。

一時パスワードには有効期限を設け、長期間使い回されないようにします。

## 情報の利用目的

保存した情報は、以下の目的で利用します。

- `/odai` 実行時にお題を出すため
- サーバーごとにカスタムお題を管理するため
- お題の連続出題を避けるため
- 管理画面へのログインを制御するため
- 誤操作や不正利用があった場合に最低限の確認を行うため

上記以外の目的で、保存情報を利用することはありません。

## 情報の公開・共有

OdaiBox が保存する情報を、第三者へ販売することはありません。

また、保存情報を広告配信や外部マーケティング目的で利用することもありません。

ただし、法令上必要な場合や、サービスの不正利用・迷惑行為への対応が必要な場合には、必要最小限の範囲で確認する可能性があります。

## 保存期間

一時パスワードやお題履歴など、短期間の利用を想定した情報は、必要な期間を過ぎたら利用しない前提で設計します。

サーバーごとのお題データは、Bot の機能を提供するために保存します。

不要になったデータについては、必要に応じて削除できるようにする想定です。

## セキュリティ上の配慮

OdaiBox では、以下を意識して設計しています。

- 固定の共通パスワードを使わない
- Discord 側の権限確認を利用する
- 一時パスワードに有効期限を設ける
- 保存する情報を機能に必要な範囲へ絞る
- 音声通話やチャット本文を保存しない
- 管理操作に関わる情報を最小限記録する

## 問い合わせ

OdaiBox は個人開発の学習プロジェクトです。

利用停止、データ削除、保存情報に関する確認が必要な場合は、Bot 管理者へ連絡してください。

## 補足

このページは、OdaiBox の情報の扱い方を説明するためのものです。

公開用ポートフォリオでは、実際のサーバー名、ユーザーID、トークン、APIキーなどの秘密情報は掲載しません。
