# OdaiBox 利用者向けガイド

OdaiBoxを友人に紹介するため、利用者向けの説明スライドを作成しました。

この資料では、ITに詳しくない利用者にも分かるように、Botの目的、基本的な使い方、管理画面でできること、保存する情報・保存しない情報、裏側の構成を説明しています。

- [利用者向け紹介スライド（PDF）](./slides/odaibox-user-slide.pdf)

## スライドで説明していること

- Discord Botとは何か
- `/odai` の使い方
- お題をWeb管理画面から追加・編集・ON/OFFできること
- 代表者だけで引く、全員が個別に引くなど、遊び方を自由に決められること
- `/admin-login` による管理画面ログイン
- 保存する情報と保存しない情報
- HTTP API方式でDiscordとLambdaを連携していること
- Gateway型を採用しなかった理由
- S3 + CloudFrontによる管理画面配信
- 小規模利用時の運用費の考え方

## ポートフォリオ上の位置づけ

このスライドは、実装した機能を非エンジニアの利用者へ説明するために作成したものです。

Botを作成するだけでなく、実際に使う人が理解しやすい形で説明資料を用意した点も、このプロジェクトで重視した内容です。
